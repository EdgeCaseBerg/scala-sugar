RegexOps

implicit final class RegexOps extends AnyVal

regex.test(str) checks for a partial match and returns a Boolean (inspired by JavaScript) This is considerably simpler than !regex.findFirstIn(str).isEmpty and semantically less awkward than regex.findFirstIn(str).isDefined.

Example: 

  1. import com.swoop.scala.sugar.Implicits._

"a+b+".r.test("xyz aabbb")
// res0: Boolean = true
